Superex is a Powerful HTML NFT marketplace template. Superex is a fully responsive, pixel-perfect, and complete template for NFTs, crypto-collectibles, digital items, non-fungible tokens websites, MetaMask Integration, and more. Superex has a lot of styles and elements to help you make a beautiful and modern website in no time. Superex is built with modern technologies like HTML5, CSS3, Pure Javascript, Bootstrap 5, and more. Superex offers various personalization options to help you give the look you require on your website. It is very easy to customize, navigate, and fits various screen sizes. Superex provides a fully responsive choice to choose LTR or RTL versions supported in both Light and Dark modes managed with CSS.
