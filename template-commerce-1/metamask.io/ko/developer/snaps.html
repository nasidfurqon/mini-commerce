MetaMask Snaps

모든 네트워크나 프로토콜의 디앱과 호환되도록 MetaMask를 확장하세요.

문서 보기문서 보기

특징

MetaMask Snaps는 개발자가 MetaMask의 기능을 안전하게 맞춤화하고 확장하여 새로운 웹3 사용자 경험을 만들 수 있도록 하는 오픈 소스 시스템입니다.

  • MetaMask에 새로운 API 메서드 추가

    Snaps를 사용하면 디앱 및 다른 Snaps와 통신하기 위한 자체 맞춤형 JSON-RPC API를 구현할 수 있습니다

  • MetaMask 사용자를 내 네트워크로 유치하기

    새로운 블록체인 네트워크에 대한 네이티브 지원을 추가하여 MetaMask 사용자를 자신의 네트워크로 유치하세요

  • 기존 MetaMask 기능 수정

    디앱과 사용자의 특정 요구사항에 맞는 새로운 사용자 경험과 상호작용을 설계하세요

  • 샌드박스 환경에서 구축하기

    사용자 데이터나 지갑 무결성을 손상시키지 않고 통제된 환경에서 새로운 기능과 기능성을 테스트하세요
