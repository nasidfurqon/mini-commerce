자주 묻는 질문

  • MetaMask 임베디드 지갑이란 무엇인가요?

    MetaMask 임베디드 지갑(구 Web3Auth)은 웹3 지갑 통합과 사용자 온보딩을 단순화하는 플러그형 지갑 인프라입니다. 앱이나 게임에 신뢰할 수 있는 MetaMask 지갑을 직접 통합할 수 있어 브라우저 확장 프로그램이 필요하지 않습니다. 사용자는 익숙한 웹2 로그인을 통해 몇 초 만에 온보딩할 수 있으며, 개발자는 사용자 경험을 완전히 제어할 수 있습니다.

  • MetaMask 임베디드 지갑은 어떤 플랫폼과 프레임워크를 지원하나요?

    MetaMask 임베디드 지갑은 모든 주요 플랫폼에 걸쳐 광범위하게 호환됩니다.
    • 웹: React, Next.js, Vue, Angular, 모든 주요 Javascript 프레임워크
    • 모바일: React Native, Flutter, iOS(Swift), Android(Kotlin)
    • 게임: Unity, Unreal Engine
  • 어떤 블록체인이 지원되나요?
    MetaMask 임베디드 지갑 SDK는 블록체인에 종속되지 않으며 모든 블록체인을 지원합니다. EVM과 솔라나 블록체인을 기본적으로 지원합니다. 또한 다른 여러 블록체인과의 통합을 위한 전용 가이드도 제공합니다.
  • 어떤 로그인 및 인증 방법을 사용할 수 있나요?
    MetaMask 임베디드 지갑은 다음을 포함한 다양한 로그인 및 인증 방법을 지원합니다.
    • 소셜 로그인: Google, Apple, Twitter, Discord 등. 모든 OAuth2 지원 로그인.
    • 비밀번호 없는 로그인: 이메일 매직 링크, 이메일 및 SMS OTP
    • 웹3 지갑: MetaMask 지갑, 모든 주요 EVM 및 솔라나 지갑
  • 기존 인증 시스템을 MetaMask 임베디드 지갑과 함께 사용할 수 있나요?
    예! 기존 OAuth2 기반 로그인을 활용하거나 기존 제공업체에 연결할 수 있습니다.
  • MetaMask 임베디드 지갑은 어떻게 통합하나요?
    MetaMask 임베디드 지갑을 통합하려면 계정을 만들고 플랜을 선택하면 됩니다. 모든 플랫폼에 대한 빠른 시작, 예제, SDK 참조를 포함한 문서를 참고할 수도 있습니다.
  • MetaMask 임베디드 지갑은 어떤 요금제가 있나요?
    무료 개발자 플랜, 다양한 기능을 갖춘 월간 활성 지갑(MAU) 기반 유료 플랜, 커스텀 엔터프라이즈 솔루션을 포함한 계층형 요금제를 제공합니다.
  • MetaMask 임베디드 지갑은 얼마나 안전한가요?
    MetaMask 임베디드 지갑(구 Web3Auth)은 임곗값 서명 방식(TSS) 및 샤미르 비밀 공유(SSS) 기반 다자간 컴퓨팅(MPC)을 포함한 고급 보안 기술을 사용합니다. 개인 키는 분할되어 노드 네트워크에 분산되므로 키가 단일 위치에 저장되지 않으며 개별 당사자가 자금에 접근할 수 없습니다. 이 아키텍처는 사용자에게 완전한 셀프 커스터디를 제공합니다.
  • MPC란 무엇이며 MetaMask 임베디드 지갑은 MPC를 어떻게 구현하나요?
    다자간 컴퓨팅(MPC)은 민감한 데이터(예: 개인 키)를 여러 암호화된 조각으로 분할하여 서로 다른 기기나 당사자에게 분산시키는 고급 암호화 기술입니다. 어떤 단일 당사자도 전체 키를 보유하지 않으므로 단일 장애 지점이 제거되고 보안이 크게 향상됩니다. 하나의 조각이 손상되어도 자산은 안전하게 유지됩니다.

MetaMask 임베디드 지갑은 임곗값 서명 방식(TSS)과 샤미르 비밀 공유(SSS)를 사용하여 MPC를 구현합니다.
    • SSS: 개인 키가 여러 조각으로 분할되며, 정해진 수 이상의 조각만 있으면 키를 복구하거나 사용할 수 있어 안전하게 백업하고 복구할 수 있습니다.
    • TSS: 여러 당사자가 키 조각을 결합하지 않고 공동으로 서명을 생성하므로 개인 키 전체가 노출될 가능성이 없습니다.
  • 스마트 지갑/계정 추상화를 어떻게 구현하나요?
    MetaMask 임베디드 지갑 SDK는 MetaMask Delegation Toolkit을 사용하는 네이티브 스마트 지갑을 제공합니다. 이는 Web3Auth 대시보드를 통해 직접 활성화할 수 있으며, User Operation Bundler를 지원하는 모든 EVM 체인에서 사용할 수 있는 간소화된 사용자 플로우, 커스터마이징 가능한 권한, 가스 없는 트랜잭션으로 다양한 기능을 구현할 수 있습니다.
  • MetaMask 임베디드 지갑은 규정을 준수하나요?
    예. MetaMask 임베디드 지갑은 GDPR, CCPA, SOC 2 Type II 표준을 준수하며 엔터프라이즈 및 글로벌 개인정보 보호 요구사항을 충족합니다. 당사의 인프라는 정기적인 감사를 받으며 99.99% 서비스 가용성을 보장합니다.
  • MetaMask 임베디드 지갑은 MetaMask 개발자 생태계와 어떻게 통합되나요?
    MetaMask 임베디드 지갑은 MetaMask 개발자 제품 라인군의 핵심으로, MetaMask SDK, Delegation Toolkit, Infura RPC와의 통합을 위한 원스톱 솔루션을 제공합니다.
  • 더 궁금한 점이 있으세요?
    커뮤니티에 참여하여 궁금한 점을 질문할 수 있습니다.
